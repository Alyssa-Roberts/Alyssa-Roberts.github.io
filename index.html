Photodiode Research

Experimental characterization and modeling of organic photodiodes, focusing on current-voltage behavior under varying bias and operating conditions. This work explored device performance, reliability, and parameter extraction using MATLAB-based modeling.

Object-Oriented Game Design in C++

Designed and implemented a console-based Mastermind game using object-oriented programming principles in C++. The project emphasized modular class design, clear game logic, and collaborative software development.

Autonomous Watering System

Built an Arduino-based automated watering system that monitored soil moisture and water levels to control a motor-driven pump. The system provided real-time feedback to the user and prioritized efficient water use for household plants.

Maze Robot

Developed an autonomous robot using an Arduino RedBoard and custom circuitry to navigate multiple maze configurations. The project involved sensor integration and iterative testing to improve navigation accuracy and stability.
